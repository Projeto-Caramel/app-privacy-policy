Política de Privacidade - Caramel

1. Introdução

O aplicativo Caramel valoriza a privacidade e a proteção dos dados pessoais de seus usuários. Esta Política detalha como coletamos, utilizamos, armazenamos, compartilhamos e protegemos suas informações.

2. Dados Coletados

Coletamos apenas os dados essenciais para o funcionamento do aplicativo e prestação dos serviços:

  • Informações Pessoais: Nome completo, e-mail, telefone, etc.
  • Dados de Perfil: Tipo de residência, estilo de vida, experiência com pets, etc.
  • Dados de Navegação: Endereço IP, logs de acesso, entre outros.

3. Finalidades do Tratamento de Dados

Os dados coletados são utilizados para:

  • Personalizar recomendações de adoção usando inteligência artificial.
  • Facilitar o contato com ONGs parceiras.
  • Gerenciar e melhorar o aplicativo.

4. Compartilhamento de Dados

Compartilhamos dados somente quando necessário, como:

  • Com ONGs parceiras para viabilizar o processo de adoção.
  • Com prestadores de serviços que auxiliam na operação do aplicativo.
  • Em conformidade com obrigações legais.

5. Armazenamento e Proteção de Dados

Implementamos medidas de segurança, como criptografia, controle de acesso e auditorias regulares, para proteger seus dados pessoais.

6. Direitos do Usuário

Você tem o direito de acessar, corrigir, excluir e solicitar informações sobre seus dados pessoais, conforme garantido pela LGPD.

7. Alterações na Política de Privacidade

Reservamo-nos o direito de modificar esta Política de Privacidade. Notificaremos você em caso de mudanças significativas.

8. Fale Conosco

Caso tenha dúvidas, entre em contato conosco pelo e-mail: contato@projetocaramel.com.br.
